Concepts Overview

  • Image: A read-only template used to create containers.
  • Container: A running instance of an image, isolated and executable.
  • Network: Connects containers so they can communicate.
  • Volume: Persistent storage that containers can use and share.
  • Registry: A server to store and distribute Docker images (like Docker Hub).
